# Session 05 — Key Phrases for Data Analysis
**Date:** 2026-06-11  
**Scenario:** Practicing key phrases for presenting data analysis — with tech lead Marcus  

---

## Conversation

**Marcus:** Let's practice some key phrases you'll use every day in remote work. First one: "Yesterday I worked on the Northwind dataset." Can you repeat that?

**Me:** Yesterday I worked on the Northwind dataset.

**Marcus:** Great! Now this one: "I found interesting patterns in the data."

**Me:** I found interesting patterns in the data.

**Marcus:** Excellent! Last one: "I would recommend analyzing this further."

**Me:** I would recommend analyzing this further.

**Marcus:** Perfect! Those three phrases will cover most of what you need when presenting analysis in a professional setting.

---

## Key phrases
1. **"Yesterday I worked on the [dataset name]."** — For standups and updates
2. **"I found interesting patterns in the data."** — For presenting findings
3. **"I would recommend analyzing this further."** — For giving recommendations
