# English Practice Log 🇺🇸

Daily English conversation practice for remote work in Data Analytics & Design.

## Goal
Build confidence for remote work conversations in English — meetings, interviews, presenting analysis, design reviews.

## Structure
Each session is a markdown file in this folder:
```
english-practice/
├── README.md          ← this file
├── session-01.md      ← first practice session
├── session-02.md      ← and so on...
```

## Weekly focus
| Week | Focus |
|------|-------|
| 1 | Job interviews & self-introduction |
| 2 | Daily standups & team communication |
| 3 | Presenting data analysis & dashboards |
| 4 | Design reviews & giving/receiving feedback |

## Key phrases to remember
- "Could you repeat that?" — cuando no entendés
- "Let me think about that for a second." — para ganar tiempo
- "As you can see in the data..." — para presentar análisis
- "I would recommend..." — para dar insights
- "I'm currently working on..." — en standups
- "Additionally / Also" — en vez de "too" al final
